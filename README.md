SmartEvent

Páginas:

1 Home: pagina principal que leva para as outras como favoritar, eventos e perfil
2 Eventos: pagina que mostra os eventos que você criou
3 Favorito: ao favoritar o evento na página anterior, nessa mostrará os eventos favoritos
4 Descrição: quando você cria um evento você tem que descreve-lo, essa página é para ver a  descrição que você criou
5 Perfil: página que mostra quem esta usando o APP, é editavel
